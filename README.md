# Markdown to HTML Converter 📝➡️🌐

A simple CLI tool to convert Markdown (`.md`) files to HTML (`.html`) using Python.

## 📦 Requirements
- Python 3.8+
- `markdown` package

Install with:
```bash
pip install -r requirements.txt
```
## How to run it
```bash
python converter.py sample.md sample.html
```
